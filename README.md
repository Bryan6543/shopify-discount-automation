<h1>AI-Powered Shopify Discount Automation Platform</h1>

<h2>Introudciton</h2>
<p>This Application Automates Shopify Discount Creation using AI. Users can generate, confirm and manage discount codes or automatic discounts, email awareness to subscribers through Mailjet and monitor store Collections all through a clean professional user interface.
</p>

<h2>Core Features</h2>
<ul>
<li>AI Generated Discounts</li>
	Auto-generate discount settings with natural language commands. (NLP)
<li>Shopify Integration</li>
	Seamless discount creation for your Shopify store.
<li>Discount Types</li>
	Supports Discount Codes and Automatic Discounts.
<li>Target Collections</li>
	Apply Discounts to specific product collections easily.
<li>Mass Email Announcements</li>
	 Notify your customer via Mailjet email integration.
<li>Confirm Before Create</li>
	 Confirmation step before creating any discount.
<li>Manage Email Lists</li>
	Add, remove and search emails through a friendly UI.
<li>Real-Time Status</li>
	View Shopify, OpenAI and Mailjet connection status live.
</ul>
